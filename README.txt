
This is a test for sharing Cinder++ projects for Xcode via git on github. 

To run: 

1. Clone Cinder from github and compile following instructions on http://libcinder.org/docs/welcome/GitSetup.html. Make sure you have downloaded boost and compiled the cinder object lib. 

2. Create a folder inside the main Cinder folder, calling it anything you like (for example: "projects").

3. On the Terminal, cd to the newly created projects folder

4. On the terminal, execute: git clone https://iani@github.com/iani/simplecinderprojectrepotest.git

5. To to folder testproject1, find and open xcode/testproject1.xcodeproj with XCode and Build / Run the project
